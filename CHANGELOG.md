## [1.0.0] - 2026-02-26

### Added
- Initial release
- GitHub Flavored Markdown rendering with full GFM support
- Syntax highlighting for 40+ programming languages
- Live file watching with automatic reload on save
- Dark/light mode following macOS system appearance
- CLI support: `mdpreview path/to/file.md`
- Drag-and-drop file opening
- File > Open (Cmd+O) menu support
- Scroll position preservation on reload
- External link handling (opens in default browser)
- Ad-hoc code signing for local use
- Makefile-based build system with install/CLI targets
